# 게임 AI 탐색 알고리즘 - 포트폴리오

## 프로젝트 개요

이 포트폴리오는 게임 AI에 사용되는 다양한 탐색 알고리즘을 C++로 구현하고 최적화한 연구 프로젝트입니다. 시간과 공간 효율성을 고려한 다양한 최적화 기법을 적용하여 실제 게임 환경에서 사용 가능한 AI 솔루션을 개발했습니다.

### 연구 목표

1. 다양한 게임 환경에 적합한 AI 탐색 알고리즘의 구현 및 비교
2. 성능 최적화 기법의 효과 검증
3. 게임 상황에 따른 최적 알고리즘 선정 기준 확립
4. 실시간 게임 AI를 위한 효율적인 의사결정 메커니즘 개발

## 구현 환경 및 게임

본 프로젝트에서는 다음 환경을 구현하여 각 알고리즘의 성능을 테스트했습니다:

1. **Maze**: 단일 에이전트가 최대 점수를 획득하는 기본 미로 환경
2. **WallMaze**: 벽이 있는 복잡한 미로에서 경로 탐색 성능 테스트
3. **AutoMaze**: 자동으로 최적의 캐릭터 배치를 찾는 메타 최적화 문제
4. **TwoMaze**: 두 플레이어가 경쟁하는 턴제 미로 게임
5. **SimMaze**: 두 플레이어가 동시에 행동을 결정하는 미로 게임
6. **Connect Four**: 실제 보드게임인 커넥트 포에 알고리즘 적용

## 구현 알고리즘

### 단일 에이전트 알고리즘
- **컨텍스트 있는 탐색**: Beam Search, Greedy, Chokudai Search
- **컨텍스트 없는 탐색**: Hill Climbing, Simulated Annealing, Random Search

### 경로 탐색 알고리즘
- **A*** (다양한 휴리스틱 함수 실험)
- **BFS** (너비 우선 탐색)
- **DFS** (깊이 우선 탐색)
- **Dijkstra** (가중치 그래프 최단 경로)

### 게임 트리 탐색
- **Minimax** (기본 게임 트리 탐색)
- **Alpha-Beta Pruning** (가지치기 최적화)
- **Iterative Deepening** (시간 제약 환경에 적합)
- **Monte Carlo Tree Search** (MCTS)
- **Thunder** (MCTS의 변형)
- **DUCT** (Decoupled UCT, 동시 행동 게임용)

### 최적화 기법
- **비트보드** (비트 연산을 통한 고속화)
- **Zobrist 해싱** (중복 상태 제거)
- **평가 함수 개선** (Value-Based 접근법)

## 주요 기술적 도전과 해결책

### 1. 효율적인 상태 표현
**도전**: 게임 상태를 메모리 효율적으로 표현하면서 빠른 연산 지원
**해결책**: 비트보드 구현과 Zobrist 해싱을 통해 해결. WallMaze에서 비트셋 최적화로 거리 계산 시간을 97.09% 단축

### 2. 실시간 의사결정
**도전**: 제한된 시간 내에 최적의 의사결정
**해결책**: 반복 심화 탐색과 시간 기반 중단 메커니즘 구현. 서로 다른 시간 제약에서의 성능 비교를 통해 최적 접근법 확인

### 3. 동시 행동 모델링
**도전**: 두 플레이어가 동시에 행동하는 환경에서의 의사결정
**해결책**: DUCT와 같은 특화 알고리즘 구현. SimMaze에서 73.5%의 승률로 가장 효과적임을 확인

### 4. 휴리스틱 함수 설계
**도전**: 효과적인 탐색을 위한 적절한 휴리스틱 설계
**해결책**: Value-Based A* 개발을 통해 단순 거리 기반보다 더 효과적인 성능 달성

## 핵심 발견점

- **알고리즘 선택의 중요성**: 게임 환경에 따라 최적 알고리즘이 크게 달라짐을 정량적으로 확인
- **비트 연산의 강력함**: 비트보드 구현으로 80.92%의 전체 실행 시간 단축 달성
- **시뮬레이션 기반 접근법의 강점**: Duct, MCTS와 같은 시뮬레이션 기반 알고리즘이 복잡한 환경에서 우수
- **휴리스틱 함수의 영향**: Value-Based A*에서 보듯 적절한 휴리스틱이 성능에 결정적 영향을 미침

## 주요 벤치마크 결과

다음은 핵심 벤치마크 결과의 하이라이트입니다:

- **Beam Search**: WallMaze에서 99.00점, 실행시간 5.02ms로 최고 성능 (점수 효율 19.72)
- **Duct 알고리즘**: SimMaze에서 73.5% 승률로 최고 성능 달성
- **Thunder 알고리즘**: AlphaBeta와의 1:1 대결에서 100% 승률 기록
- **비트셋 최적화**: 실행 시간 80.92% 단축 (5.24ms → 1.00ms)
